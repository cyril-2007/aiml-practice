# MNIST Handwritten Digit Classifier

A Deep Learning project built using TensorFlow and Keras that recognizes handwritten digits (0–9).

## Features

- Trains a neural network on the MNIST dataset
- Achieves around 97–98% test accuracy
- Saves the trained model
- Uses TensorFlow/Keras

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Output

- Downloads the MNIST dataset
- Trains the neural network
- Evaluates model accuracy
- Saves the trained model to `saved_model/mnist_model.keras`
