# 🌸 Iris Flower Classifier

## Overview
This project predicts the species of an Iris flower using Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- NumPy

## Machine Learning Algorithm
- Decision Tree Classifier

## Features
- Loads the built-in Iris dataset
- Splits data into training and testing sets
- Trains a Decision Tree model
- Predicts flower species
- Displays model accuracy
- Shows confusion matrix
- Shows classification report

## Files
main.py          - Predict flower species
evaluation.py    - Evaluate model performance

## How to Run

```bash
python main.py
```

or

```bash
python evaluation.py
```

## Example Output

