# Face Detector

A simple AI project built using OpenCV that detects human faces in an image using a Haar Cascade Classifier.

## Features
- Detects faces in an image
- Draws rectangles around detected faces
- Displays the number of detected faces

## Technologies Used
- Python 3.13
- OpenCV 4.10.0

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
